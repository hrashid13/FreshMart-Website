# FreshMart Website

This website is entirely designed, written, and maintained by AI agents. No human writes the code or content.

[![Live Demo](https://img.shields.io/badge/Demo-FreshMart-green)](https://hrashid13.github.io/FreshMart-Website/)

## How it works

Three AI agents run automatically every Monday via GitHub Actions. Each agent makes a call to the Claude API in sequence — no local infrastructure required.

**Idea Agent** — Invents the weekly ad from scratch. It creates the weekly theme, sale items with prices, a seasonal recipe, and a store announcement. It has no access to any real inventory data. Everything is generated.

**Coder Agent** — Receives the Idea Agent's output and writes the `index.html` for the site. It structures the content into semantic HTML with a header, deals section, recipe section, and announcement.

**Style Agent** — Writes the matching `style.css`. It handles layout, color scheme, typography, and responsive design.

Once all three agents finish, the pipeline commits the files to a feature branch and opens a pull request on this repository. A human reviews and merges the PR to publish the update.

## Deployment

The site is hosted on GitHub Pages and deploys automatically from the `main` branch. Every merge to `main` triggers a new deployment.

## Repository structure

```
index.html                       — generated by the Coder Agent
style.css                        — generated by the Style Agent
runner.py                        — orchestrates the three agents via Claude API
publish.py                       — pushes the branch and opens the pull request
requirements.txt                 — Python dependencies
.github/workflows/weekly.yml     — GitHub Actions cron job (every Monday 8am)
```

## Pull request history

Every weekly update exists as a merged pull request in this repository. The PR history is a complete record of every change the agents have ever made, including the commit message, the diff, and the date it was published.

## Tech stack

- Claude API (claude-sonnet) — AI agent inference
- GitHub Actions — automated weekly scheduling
- GitHub Pages — static site hosting
- GitHub API — automated pull request creation
